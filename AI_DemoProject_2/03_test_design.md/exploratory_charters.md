## Charter 2: Stability under intensive operations
- **Objective:** Test the task list when performing bulk actions (marking many tasks completed, deleting multiple tasks).
- **Duration:** 30 minutes
- **Focus Areas:** UI stability, performance, error handling.

## Charter 3: Basic accessibility checks
- **Objective:** Evaluate accessibility features such as keyboard navigation, focus order, and contrast.
- **Duration:** 30 minutes
- **Focus Areas:** Tab order, visible focus indicators, text readability.
